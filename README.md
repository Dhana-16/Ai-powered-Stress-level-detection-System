# AI-Powered Stress Level Detection System

An intelligent AI/ML-based system that detects and analyzes human stress levels using user data and machine learning models.

## Features
- Real-time stress level prediction
- Interactive dashboard with charts
- AI/ML-based analysis
- User-friendly UI
- Secure data handling

## Tech Stack
- Frontend: HTML, CSS, JavaScript
- Backend: Python (Flask/Django)
- Machine Learning: Scikit-learn / TensorFlow
- Visualization: Chart.js
- Database: SQLite / MongoDB

## How It Works
1. User enters personal and behavioral data
2. AI model processes inputs
3. Stress level is predicted
4. Results shown in dashboard
